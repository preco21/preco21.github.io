This branch contains source of [prevl.dev](https://prevl.dev) website.

## Install

```bash
$ git clone https://github.com/preco21/preco21.github.io.git
$ cd preco21.github.io
$ yarn
```

## Usage

### Serve website

```bash
$ yarn start
```

### Build website

```bash
$ yarn run build
```

### Deploy (on the `master` branch)

```bash
$ yarn run deploy
```
